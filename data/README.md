# Data Notes

Use the **Superstore** dataset (commonly used in Tableau demos). You can download one of these:
- Kaggle: "Superstore Dataset" (CSV/Excel)
- Tableau Sample Data: "Sample - Superstore"

> Place the file under: `data/superstore.csv` (or `.xlsx`)  
> Then follow the build steps in `../tableau/build_instructions.md`.

**Expected fields (examples):**
- Order Date, Ship Date
- Category, Sub-Category, Product Name
- Sales, Quantity, Discount, Profit
- Country, State/Province, City, Postal Code
